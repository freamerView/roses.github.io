# Розы Брянск — лендинг доставки роз

Лендинг доставки роз и букетов по Брянску. Собран по той же структуре, что и основной лендинг проекта: hero, каталог, преимущества, отзывы, формы, корзина.

## Структура

- **index.html** — главная: hero, блок «Почему мы», каталог с фильтрами, отзывы, форма «Подобрать букет», корзина (сайдбар)
- **about.html** — о компании
- **contacts.html** — контакты, карта
- **policy.html** — политика конфиденциальности
- **offer.html** — публичная оферта
- **css/styles.css** — стили (палитра розовый/бордовый)
- **js/data.js** — каталог букетов (PRODUCTS)
- **js/main.js** — корзина, фильтры, модалка товара
- **js/seo.js** — canonical и og:url

## Что подставить под себя

1. **Формы** — в `index.html`, `about.html`, `contacts.html` в атрибутах `action` форм указан Formspree (`https://formspree.io/f/xeqwknqz`). Замените на свой endpoint Formspree или другой обработчик.
2. **Картинки** — в каталоге используются пути `images/catalog/1.jpg` … `images/catalog/12.jpg` и `images/hero.jpg`. При отсутствии файла показывается SVG-заглушка. Добавьте свои фото в `images/catalog/` и `images/hero.jpg`.
3. **Контакты** — телефон, Telegram, email, адрес в `contacts.html` и в schema.org в `index.html` заданы условно — замените на реальные.

## Запуск

Откройте `roses/index.html` в браузере или поднимите локальный сервер из папки `roses` (чтобы пути к CSS/JS работали корректно при открытии через file://).
